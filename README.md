# Tic Tac Toe


Simple Tic Tac Toe game for Terminal built in Elixir

### Interface

```bash
Where do you want to place your piece (o)? 
> A2
      A     B     C
   |-----|-----|-----|
   |     |     |     |
 1 |  x  |     |     |
   |     |     |     |
   |-----|-----|-----|
   |     |     |     |
 2 |  o  |  o  |  x  |
   |     |     |     |
   |-----|-----|-----|
   |     |     |     |
 3 |     |     |     |
   |     |     |     |
   |-----|-----|-----|
```

### Usage

```
$ git clone https://github.com/at1as/tic-tac-toe
$ iex play.ex 
> Game.play
```
